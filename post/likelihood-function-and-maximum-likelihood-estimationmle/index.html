Likelihood Function and Maximum Likelihood Estimation(MLE)

2021-12-14 3 min read # Math

Example: Given 3 data points 9,9.5 and 11, which are very likely coming from a Normal distribution, MLE can be used then to give the estimation of the most likely parameters of this Normal distribution.

P(x;μ,σ)=1σ2πexp(−(x−μ)22σ2)P(x;\mu,\sigma) = \frac{1}{\sigma\sqrt{2\pi}}exp(-\frac{(x-\mu)^2}{2\sigma^2}) P(x;μ,σ)=σ2π​1​exp(−2σ2(x−μ)2​)

The joint probability density of the three data points is given by:
P(9,9.5,11;μ,σ)=1σ2πexp(−(9−μ)22σ2)∗1σ2πexp(−(9.5−μ)22σ2)∗1σ2πexp(−(11−μ)22σ2)P(9,9.5,11;\mu,\sigma) = \frac{1}{\sigma\sqrt{2\pi}}exp(-\frac{(9-\mu)^2}{2\sigma^2}) * \frac{1}{\sigma\sqrt{2\pi}}exp(-\frac{(9.5-\mu)^2}{2\sigma^2}) * \frac{1}{\sigma\sqrt{2\pi}}exp(-\frac{(11-\mu)^2}{2\sigma^2}) P(9,9.5,11;μ,σ)=σ2π​1​exp(−2σ2(9−μ)2​)∗σ2π​1​exp(−2σ2(9.5−μ)2​)∗σ2π​1​exp(−2σ2(11−μ)2​)
By taking the differentiation of this joint probability function and make it equal to 0, we can obtain the maximum value of it and μ,σ\mu,\sigma μ,σ

Log likelihood is used because the differentiation above is difficult to calculate and the natural logarithm is a monotonically increasing function which is fit for our purpose.

Applying logarithms gives us a simplified probability:
lnP(x;μ,σ)=−3ln(σ)−32ln(2π)−12σ2[(9−μ)2+(9.5−μ)2+(11−μ)2]ln P(x;\mu,\sigma) = -3ln(\sigma) - \frac{3}{2}ln(2\pi) - \frac{1}{2\sigma^2}[(9-\mu)^2+(9.5-\mu)^2+(11-\mu)^2] lnP(x;μ,σ)=−3ln(σ)−23​ln(2π)−2σ21​[(9−μ)2+(9.5−μ)2+(11−μ)2]

Taking the partial derivative of μ\mu μ, we have:
∂lnP(x;μ,σ)∂μ=1σ2[9+9.5+11−3μ]\frac{\partial ln P(x;\mu,\sigma)}{\partial \mu} = \frac{1}{\sigma^2}[9 + 9.5 +11 - 3\mu] ∂μ∂lnP(x;μ,σ)​=σ21​[9+9.5+11−3μ]

Setting the derivative to zero and we have μ=9+9.5+113=9.833\mu = \frac{9+9.5+11}{3} =9.833 μ=39+9.5+11​=9.833

In real-world scenario the derivative of log-likelihood is still hard to solve, we can then apply iterative methods like Expectation-Maximisation algorithms to find numerical solutions for the parameter estimates.
